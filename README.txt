== GUIDE D'UTILISATION ==

1. Ouvre le dossier "PlateformeJeux" dans Visual Studio Code
2. Place tous tes fichiers Java (comme Membre.java, Jeu.java, etc.) dans :
     src/et3projet/
   (Ils doivent commencer par : package et3projet;)
3. Clique sur le bouton "Run" ou va dans l'onglet "Run and Debug"
4. Sinon, utilise le terminal :
     cd src
     javac et3projet/Plateforme.java
     java et3projet.Plateforme
